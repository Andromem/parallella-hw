Porcupine
=================================

## Summary

* PARALLELLA BREAKOUT BOARD
* UART
* I2C (5V)
* SPDIF
* ANALOG INPUT
* PMOD
* JTAG
* 0.1" GPIO HEADERS WITH ADJUSTABLE VOLTAGE
* RASPBERRY PI CAMERA INTERFACE
* ELINK HEADERS
