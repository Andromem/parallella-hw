## Summary
* Meta Module with 256 RISC CPU (4x4 grid of E16G301 chips)
* meta Carrier with programmable power, clocks, FMC connector
* Up to 16 elinks per module
* PCI/104 form factor (3.775" x 3.55")
* <32 Watt per module
